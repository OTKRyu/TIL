## 폴더(directory) 관련

폴더 상징 특수기호

- `~`
  - Home directory
  - /c/users/(name)
- `/`
  - Root directory()
- `.`
  - 현재 위치
- `..`
  - 상위 폴더

## 명령어

- `cd[대상폴더]`
  - 대상폴더로 이동
- `ls`
  - 폴더 내부 항목 리스트업
- `rm`
  - 삭제
  - -r dirctory 삭제
  - -a 하위내용 전부 삭제
  - -f 강제 삭제
- `mkdir'
  - 디렉토리 생성
- `touch`
  - 생성
